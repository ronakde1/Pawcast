This app is built using the Expo React Native Framework. Weather forecast data is obtained from the https://open-meteo.com/ API.

Environment set up dependency requirements:
- node
- npm

Steps to run the code:

1. From inside the pawcastapp directory, run npm install to install all the require node packages
2. Run npx expo start to run the app 
3. To run the app on a mobile device, download the Expo Go (https://expo.dev/go) app
4. Use the QR scanner to scan the QR code generated from the Step 2. The expo app will then open on the mobile device.
